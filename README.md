# VYDA Restaurant Booking-nettside

Dette er et **gruppeprosjekt i emnet IT2805 Webteknologi** ved NTNU, 1. på informatikk. Oppgaven gikk ut på å lage en enkel og funksjonell nettside av fritt valg.  
Vi valgte **VYDA Restaurant**, en vietnamesisk restaurant ved Valentinlyst, som inspirasjon.

## Om prosjektet

**VYDA Restaurant** bruker i dag kun Facebook, og har ingen offisiell nettside.  
Vi ønsket derfor å lage en *fiktiv, men realistisk* nettside for dem, med fokus på:

- Et moderne og ryddig design  
- En tydelig meny-side  
- Enkel kontaktinformasjon og lokasjonsvisning  
- En reservasjonsside (simulert med HTML-skjema)  

Dette prosjektet ble avsluttet ved slutten av semesteret og vil ikke bli videreutviklet.

## Teknologier brukt

- **HTML5**  
- **CSS3**  
- Enkel **JavaScript** for layout og interaktivitet  

Ingen backend eller database ble brukt — prosjektet er en **statisk nettside**.

## Hvordan kjøre prosjektet lokalt

1. Last ned eller klon prosjektet
2. Åpne prosjektmappen i Visual Studio Code.
3. Installer utvidelsen Live Server (hvis du ikke allerede har den installert).
4. Høyreklikk på main_page.html → velg "Open with Live Server".
5. Nettleseren åpnes automatisk og viser nettsiden
